# Sunshine Motor Driving School Website

Professional website for Sunshine Motor Driving School, Malviya Nagar, Jaipur. Established 2007.

## Files

- `index.html` — Main HTML structure
- `style.css` — All styles (variables, layout, responsive breakpoints)
- `script.js` — Form handler, gallery lightbox, scroll-to-top, smooth nav
- `assets/` — Logo, training car images, WhatsApp photos

## Features

- Mobile-first responsive design (380px → 1200px+)
- Hero enquiry form → WhatsApp integration (`wa.me/918502010101`)
- Course cards (Hatchback / Sedan / Compact SUV) with pricing
- FAQ accordion, gallery lightbox, scroll-to-top button
- JSON-LD structured data for SEO
- Open Graph & Twitter Card meta tags
- Local-only images (zero external hosting)

## External Dependencies (CDN)

- Bootstrap 5.3.0
- Bootstrap Icons 1.10.5
- Font Awesome 6.5.0
- Google Fonts: Barlow + Inter Tight

## Setup

Open `index.html` in a browser. No build step required.
